# 매드업(MADUP) 퍼포먼스 마케팅 제안서 생성 Skill

## Skill 목적
매드업의 실제 제안서 5종(2026 빗썸, 2026 TVING, 2025 와이어트, 2026 삼성카드, 2026 삼성증권)을 분석한 시그니처 구조·톤·자체 솔루션을 사용해 디지털 퍼포먼스 마케팅 제안서를 자동 생성합니다.

## 선행 작업 권장
제안서 작성 전 `fact-book` skill로 팩트북을 먼저 작성하면 품질이 크게 향상됩니다. 팩트북 디렉토리(`/Users/gimjuhyeon/Desktop/제안서_자동화/[광고주명]_팩트북/`)가 있으면 이 skill이 자동으로 참조합니다. 팩트북이 없으면 사용자에게 fact-book skill 선행 실행을 권장하되, 사용자가 원하면 사용자 제공 정보만으로도 진행 가능.

## 사용 시점
- 신규 RFP 응답용 제안서 초안 작성
- 디스커버리 콜 이후 제안 공식화
- 기존 광고주의 연간 계획(YoY) 제안서 작성
- 트리거 문구: "퍼포먼스 마케팅 제안서", "[광고주명] 제안서", "DA/SA 제안서", "매드업 제안서 작성", "마케팅 제안서"

- **LEVER Xpert**: 통합 광고 자동화 플랫폼 — 4개 센터
  - 자산 관리 & 데이터 센터: 광고 계정 연동, 데이터 수집/전처리 자동화, 리포트 자동화, API 미지원 데이터 연동

  - 운영 센터: 시간/날씨/조건별 상태·예산 변경, 피드 메이커(카탈로그 광고용 피드 커스터마이징), 매체 캐시 관리
  - 트랜드 센터: 카테고리 소재 아카이빙 + AI 패턴 분석 + AI 기획안 생성 (PPT/PSD/Figma 편집 가능)

- **PRISM**: 광고 운영 통합 솔루션 (Operation Step)
- **자동입찰**: 성과기반 입찰 자동화 (1순위 마킹 입찰 한계를 극복하는 실시간 데이터 기반)
- **MADUP ON-AIR**: 고객사/경쟁사 모니터링 + 리스크 실시간 대응

- **AI Creative 생성 툴**: 미드저니 + 자체 모델 커스터마이징, AI 영상/소재 자동 생성

## 제안서 구조 — 4가지 변형 (광고주/캠페인 성격에 맞춰 선택)

| 챕터 | 내용 | 분량 비중 |
|---|---|---|
| Ⅰ | 제안 개요 | 5% |
| Ⅱ | 제안사 현황 (회사 소개, 조직, 파트너십, 수상, 고객사, 수행실적) | 10% |
| Ⅲ | 시장 현황 (거시 진단, 통계적 근거) | 10% |
| Ⅳ | 채널/매체 운영전략 및 KPI 설정 (시그니처 프레임워크 + DA/SA/크리에이티브) | 45~50% |
| Ⅴ | 제공 솔루션 (LEVER Xpert, MILOT, AI, 외부) | 10% |
| Ⅵ | 분석 보고 (대시보드, 예측 모델링, MMM) | 5% |
| Ⅶ | 전담 인력 구성 (조직도 + R&R 표) | 5% |

| 끝 | "감사합니다" + Copyright | 1p |

챕터 표기: 로마숫자(Ⅰ~Ⅷ). 푸터: `[챕터].[하위주제] | [세부주제] [페이지번호]`

- 표지 양식: `[광고주] [연도] 퍼포먼스 마케팅 제안서 / Copyright + Tel + 주소`

### 변형 C. 브랜드 캠페인형 (브랜드 빌드업 · D2C · 신상품 런칭)
**예시**: 와이어트(77p)

내러티브 6단계:
1. **기회 발견** — 시장 진입 컨텍스트
2. **1차 성과** — 기존 제품/액션의 성공 검증
3. **2차 액션** — 확장/전환 의사결정
4. **문제 발생** — 새로운 시장에서의 한계
5. **근본 원인 분석** — 데이터 기반 진단
6. **최종 결론 + 매드업 솔루션**

구성:
- **현황 분석 및 과제 도출** (3-way 분석: 시장 + 경쟁사 + 자사) — 30~40%
- **유사 사례 인용** (예: 메디힐 마스크팩 → 토너 패드 확장 사례) — 5%
- **매드업 제시 컨셉** — 핵심 전략 페이지 5~10p
- **커뮤니케이션 · 크리에이티브 map** (직관적/감각적/서정적/스토리적 × 메시지/프로모션/시즈널/CTA) — 30%
- **매체 운영 + 리포트** (데일리/월간) — 15%
- **조직 구성안 및 수수료 제안** — 5%
- **Special Offer** (수수료율, Tech 솔루션 무상, 소재 제작 무상, 크레딧 등) — 5%

페이지 푸터: 모든 페이지에 "DIGITAL MARKETING GROUP MADUP" 워드마크 고정

### 변형 D. 갱신/전략 업데이트형 (기존 광고주 · 연간 갱신 · 솔루션 추가 제안)

특징:
- **분량 압축** (50~80p) — 회사 소개·매체 파트너십·수상 이력 등은 생략 또는 부록으로 (광고주가 이미 알고 있음)
- **솔직한 자기 평가** — "광고 효율은 지속 개선되었으나, MAU 같은 실질 점유는 다소 보합세" 같은 한계 인정
- **3년 운영 타임라인** + 거시경제/시장 변화 진단
- **신규 솔루션 제안 중심** (GEO, 인블로그, 더블리스팅 등 매드업 신상 기술)

표준 구성:
1. **상황 진단 및 전략** (전체 약 25%)
   - 거시 시장 변화 (예: 트럼프 집권 후 예측 불가, IMA/발행어음, 경쟁사 부상)
   - 광고주 시장 점유 변화 (MAU 추이 등)
   - **3년 매드업 X [광고주] 광고 효율 및 주요 액션** (23년·24년·25년 액션 + CPA 추이 그래프)
   - **MADUP이 진단한 원인 / 우리를 성공으로 이끌었던 그것의 한계**
2. **새 프로토콜/전략 #1** (예: "眞 신규 모객 프로토콜" 3단계) — 약 30%
3. **새 프로토콜/전략 #2** (예: "어려운 상품 모객 방안") — 약 20%
4. **새 솔루션 (For True SOV / GEO 등)** — 약 15%
5. **Summary + 미디어 전략 지도** (매트릭스 표) — 5%
6. **Appendix**: 데이터 운용 변화, 인력 R&R, 디자인 단가표 — 5%
7. 마지막: "감사합니다. For Real Growth"

페이지 헤더: 모든 페이지에 광고주명 영문 "[Client] Digital Marketing Proposal" 고정
